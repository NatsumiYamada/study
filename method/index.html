カレー関数でカレーを作ってみる

画像をクリックするとカレーが完成します
画像の違いでできるカレーが変わります

何カレーができるかな

カレー関数内容

やることは画像から取得した引数からテキスト、画像のsrcを作成するだけです。

    //HTMLのidから取得
    var text  = document.getElementById("text");
    var curryPNG  = document.getElementById("curryPNG");

    //カレー関数
    function makeCurry(meet){
        //テキストを表示
        text.innerHTML = meet + 'カレーが完成しました！';
        //画像を表示
        var srcPNG;
        switch (meet){
            case '豚肉':
                srcPNG = 'img/pig_curry.png';
                break;
            case '野菜':
                srcPNG = 'img/begi_curry.png';
                break;
            default:
                srcPNG = 'img/dish.png';
                break;
        }
        curryPNG.src = srcPNG;
    }

関数とはサブプログラム

  • 関数とはメインのプログラムに紐づくプログラム
  • 関数の命名は手順化した作業に名前をつけること
  • 引数は関数で使う値を渡す

カレー関数に置き換えると

関数はカレーを作るレシピのこと、カレーの作り方がカレー関数に書かれます。
引数は具材のことで、豚肉やチキンのことです。豚肉を使ったら豚肉カレーができ、
野菜を使用したら野菜カレーができます。
ここで、カレーのレシピであるカレー関数の内容は変わりません。
